# 🧠 Scratch Implementations

This folder contains Machine Learning algorithms implemented from scratch using NumPy.

---

## 📌 Purpose

* Understand core mathematical concepts behind ML algorithms
* Implement learning algorithms without relying on libraries
* Build intuition for optimization, gradients, and distance metrics

---

## ⚙️ Implemented Algorithms

* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* K-Means Clustering
* Principal Component Analysis (PCA)

---

## 🧩 Key Concepts Covered

* Gradient Descent
* Cost Functions (MSE, Binary Cross-Entropy)
* Distance Metrics (Euclidean)
* Clustering & Dimensionality Reduction

---

## ▶️ Example

```bash
python main.py
```

Switch mode in `main.py`:

```python
mode = "scratch"
```
