# 🛠️ Utilities

This folder contains reusable components used across different ML pipelines.

---

## 📦 Modules

### `preprocessing.py`

* Handle missing values
* Encode categorical features
* Feature engineering

---

### `evaluation.py`

* Accuracy, Precision, Recall, F1 Score
* Confusion Matrix
* ROC-AUC calculation

---

### `visualization.py`

* ROC Curve plotting
* Model performance visualization

---

## 🎯 Purpose

* Avoid code duplication
* Keep pipelines clean and modular
* Improve reusability across projects
