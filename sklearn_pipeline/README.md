# ⚙️ Sklearn Pipelines

This folder contains end-to-end Machine Learning workflows using scikit-learn.

---

## 📌 Purpose

* Build practical ML pipelines
* Compare multiple models on real datasets
* Apply proper evaluation techniques

---

## 🔍 Features

* Data preprocessing and feature engineering
* Model training using:

  * Logistic Regression
  * KNN
  * Decision Tree
  * Random Forest
* Model comparison using multiple metrics
* ROC Curve and AUC evaluation
* Cross-validation for robustness

---

## 📊 Datasets Used

* Titanic → Classification
* Iris → Multi-class classification
* House Prices → Regression
* Mall Customers → Clustering

---

## ▶️ Example

```bash
python main.py
```

Switch mode in `main.py`:

```python
mode = "sklearn"
```
